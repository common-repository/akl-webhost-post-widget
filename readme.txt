== Installation ==

This plugin enables you to edit and display Content Blocks in a sidebar widget or using a shortcode.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Appearance->Widgets->Post Widget (AKL Webhosting) screen to configure the plugin
4. Drag that plugin and drop whereever you want to dislay
5. Then that plguin automatically start displaying your posts
